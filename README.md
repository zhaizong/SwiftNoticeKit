# SwiftNoticeKit
GUI library for displaying various popups (HUD), written in pure Swift.

# Features

# Pretty easy to use

```swift
pleaseWait()
successNotice("Success!")
errorNotice("Error!")
infoNotice("Info!")
noticeOnlyText("noticeOnlyText")
noticeTop("abcd!")
clearAllNotice()
```

# Requirements

* iOS 8.0+
